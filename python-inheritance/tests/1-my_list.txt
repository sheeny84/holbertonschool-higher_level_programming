The ``MyList`` class module
======================

Using ``MyList``
-------------------

First import ``MyList`` from the ``1-my_list`` module:
    >>> MyList = __import__('1-my_list').MyList

Now use it:

    >>> my_list = MyList()
    >>> my_list.print_sorted()
    []
    
    >>> my_list.append(1)
    >>> my_list.append(4)
    >>> my_list.append(2)

    >>> print(my_list)
    [1, 4, 2]

    >>> my_list.print_sorted()
    [1, 2, 4]

    >>> print(my_list)
    [1, 4, 2]

    >>> my_list.append(-2)
    >>> my_list.print_sorted()
    [-2, 1, 2, 4]
